<task type="auto">
  <name>Task 2: Create GitHub Actions CI/CD workflow for GitHub Pages deployment</name>
  <files>.github/workflows/deploy.yml</files>
  <action>
1. Create `.github/workflows/deploy.yml` with the following pipeline:

   Trigger: push to main branch only.

   Permissions: contents read, pages write, id-token write (required for GitHub Pages deployment).

   Concurrency: group "pages", cancel-in-progress false (let deployments finish).

   Steps in order (BUILD ORDER IS CRITICAL -- see research pitfall #5):
   a. Checkout code (actions/checkout@v4)
   b. Setup Rust nightly with WASM target (dtolnay/rust-toolchain@nightly with targets: wasm32-unknown-unknown, components: rust-src)
      - If LLVM 20 issues occur, pin nightly date: `toolchain: nightly-2025-05-15`
   c. Install wasm-pack from pre-built binary (NOT cargo install):
      ```
      curl https://rustwasm.github.io/wasm-pack/installer/init.sh -sSf | sh
      ```
   d. Install flutter_rust_bridge_codegen: `cargo install flutter_rust_bridge_codegen`
   e. Setup Flutter (subosito/flutter-action@v2, channel: stable)
   f. Flutter pub get
   g. Generate FRB Dart bindings: `flutter_rust_bridge_codegen generate`
   h. Build WASM: `dart run flutter_rust_bridge build-web --release`
   i. Build Flutter web: `flutter build web --wasm --release --base-href "/pulijoodam/"`
      CRITICAL: --base-href MUST match the GitHub repo name with leading and trailing slashes.
   j. Copy coi-serviceworker.js to build output: `cp web/coi-serviceworker.js build/web/`
      (The Flutter build does NOT automatically copy this file)
   k. Upload Pages artifact: actions/upload-pages-artifact@v3 with path: build/web/
   l. Deploy to GitHub Pages: actions/deploy-pages@v4

2. Use a single job `build-and-deploy` with environment `github-pages`.

3. Add Rust dependency caching to speed up CI:
   ```yaml
   - uses: actions/cache@v4
     with:
       path: |
         ~/.cargo/registry
         ~/.cargo/git
         rust/target
       key: rust-wasm-${{ hashFiles('rust/Cargo.lock') }}
       restore-keys: rust-wasm-
   ```

4. Add Flutter pub cache:
   ```yaml
   - uses: actions/cache@v4
     with:
       path: ~/.pub-cache
       key: flutter-pub-${{ hashFiles('pubspec.lock') }}
       restore-keys: flutter-pub-
   ```
  </action>
