Create `src/hooks/useAIGame.ts`:

**State shape (extends UIState concept):**
```typescript
interface AIUIState {
  gameState: GameState;
  history: GameState[];
  redoStack: GameState[];
  selectedNode: number | null;
  legalMoves: LegalMove[];
  lastEvents: GameEvent[];
  isAIThinking: boolean;
  requestId: number;  // increments on each AI request; stale responses discarded
}
```

**Actions (extends UIAction):**
- All of useGame's actions: NODE_TAPPED, NEW_GAME, UNDO, REDO, END_CHAIN
- Plus: AI_THINKING (set isAIThinking=true), AI_MOVE_RECEIVED (apply AI move, set isAIThinking=false), AI_RESPONSE_DISCARDED

**Hook signature:**
```typescript
export function useAIGame(config: { humanRole: Role; difficulty: AIDifficulty })
```

**Key behaviors:**
1. **Worker lifecycle**: Create worker on mount via `new Worker(new URL('../engine/ai/worker.ts', import.meta.url), { type: 'module' })`. Terminate on unmount. Store in useRef.
2. **AI turn trigger**: useEffect watches `gameState.currentTurn` and `chainJumpInProgress`. If current turn !== humanRole AND status === 'ongoing' AND not already thinking, dispatch AI_THINKING and postMessage to worker.
3. **Response handling**: Worker onmessage checks requestId. If stale, discard. If fresh, apply MIN_AI_DELAY_MS floor (400ms minus actual think time), then dispatch AI_MOVE_RECEIVED.
4. **Undo pairing (AI-10)**: UNDO action checks if `history.length >= 3` (initial + human move + AI move). If so, step back 2 entries. If only 2 entries (initial + AI first move when human plays tiger), step back 1. Cancel any in-flight AI request by incrementing requestId.
5. **New game**: Increment requestId (cancels in-flight), reset state, DO NOT terminate/recreate worker (reuse is fine since requestId guards staleness).
6. **Return interface**: Same shape as useGame return value plus `{ isAIThinking }`. This allows GameScreen to use either hook with minimal changes.

  - Test AI_THINKING sets isAIThinking to true

- For the delay logic, test that the delay calculation is correct (400 - thinkTimeMs, clamped to 0)
