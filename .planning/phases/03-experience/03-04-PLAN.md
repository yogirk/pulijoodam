<objective>
Build game persistence (auto-save and resume) and a complete history/replay system so players never lose progress and can review past games.

Purpose: Without auto-save, closing the browser tab means losing game progress. Without history, every game is ephemeral. This plan makes the game feel like a real application that remembers the player.

Output: Auto-save on every move, automatic game resume on reopen, a history screen listing past games, and a full replay viewer with step controls, scrubber, and auto-play.
</objective>
